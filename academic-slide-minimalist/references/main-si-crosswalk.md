# Main Paper + SI Crosswalk

Use this reference to connect the main paper's story with supplementary evidence.

## Purpose

The main paper usually gives the narrative. The SI usually gives the proof details. A strong literature-report PPT needs both.

## Crosswalk Table

Maintain this mapping while reading:

```text
main-text claim | related main figure | related SI figure/table/method | what SI adds | slide use
```

## What to Search in SI

Check SI for:

- synthesis recipes, reagent ratios, temperatures, reaction times;
- instrument parameters and data-processing methods;
- extra XRD/PXRD, SEM/TEM/STEM, XPS, FTIR, Raman, NMR, BET, TGA/DSC, DFT, electrochemical or catalytic data;
- control samples and blank experiments;
- repeated trials, recycling, stability, long-term tests;
- expanded comparison tables;
- raw or uncropped data that clarifies the main figure.

## How to Use SI in Slides

- Use SI methods for “具体制备方法/实验条件” pages.
- Use SI controls for “为什么这个结论可信” pages.
- Use SI stability/recycling data for reliability pages.
- Use SI comparison data to support advantage claims.
- Keep SI figure IDs visible or noted, e.g. `Fig. S7`, `Table S2`.

## Caution

Do not overload the PPT with SI. Bring in SI only when it supports a key slide claim or explains a necessary experimental detail.
