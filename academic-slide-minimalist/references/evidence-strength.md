# Evidence Strength Labels

Use these labels to calibrate slide titles, bullet text, and speaker notes.

## Labels

### direct evidence

The figure directly shows the claimed result.

Examples:

- microscopy directly shows morphology or pore arrays;
- crystal structure refinement supports a framework model;
- a performance chart directly compares samples under stated conditions.

Suggested wording:

```text
直接观察到 / 直接显示 / 共同证实
```

Use “共同证实” only when two or more independent evidence types support the same claim.

### indirect evidence

The figure supports the claim through a proxy measurement.

Examples:

- adsorption suggests pore structure;
- spectroscopy suggests coordination or composition;
- PXRD suggests phase retention.

Suggested wording:

```text
表明 / 说明 / 支持
```

### supplementary support

The figure is in SI or supports a claim that is mainly introduced in the main paper.

Suggested wording:

```text
进一步支持 / 补充说明 / 从补充实验中可以看到
```

### preliminary evidence

The evidence is limited in scope, sample range, duration, or conditions.

Suggested wording:

```text
初步表明 / 在该条件下显示 / 提示
```

### proposed mechanism

The authors propose an explanation, but the paper does not fully prove the mechanism.

Suggested wording:

```text
作者认为 / 可能与……有关 / 该结果支持这一解释
```

Avoid:

```text
证明机理完全成立
```

### application validation

The data show functional or performance relevance.

Suggested wording:

```text
验证其应用潜力 / 体现出性能优势 / 支持其用于……场景
```

Avoid claiming industrial readiness unless the paper demonstrates real industrial deployment.

## Overclaim Control

Before finalizing a slide title, ask:

```text
Does the evidence directly prove this, or only support/suggest it?
```

If the evidence is indirect or preliminary, soften the title.
