# Post-Assembly Render Audit

Use this reference after creating the PPTX.

## Core Rule

A PPT is not final until the assembled deck has been visually audited as a whole.

## Required Audit Steps

1. Open or render the PPT pages to images when tooling permits.
2. Create a contact sheet / montage, usually 6x6 for a 36-page deck or a grid suited to the page count.
3. Check global page order.
4. Check navigation continuity and active section highlights.
5. Check page-number sequence.
6. Check style consistency across batches.
7. Check whether any old draft page, missing-navigation page, or backup-like page entered the mainline by mistake.
8. Check for blank, duplicated, distorted, or unreadable pages.
9. Fix issues and reassemble if needed.

## Output Files

When feasible, include:

```text
final_ppt_render_montage.png
post_assembly_audit.md
```
