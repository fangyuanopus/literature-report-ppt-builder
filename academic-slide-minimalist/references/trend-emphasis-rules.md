# Trend Emphasis Rules

Use this reference to make figures easier to explain in a literature-report PPT.

## Core Rule

Do not merely place a figure on a slide. Decide what the audience must notice and annotate only that.

## Required Per-Figure Fields

For every evidence slide, identify these before layout:

```text
key trend:
key number(s):
key comparison:
what to highlight:
what not to overexplain:
claim wording:
speaker sentence:
```

## Examples

### Long-term performance figure

```text
key trend: conversion/selectivity remain stable over long time
key numbers: time on stream, conversion, selectivity
what to highlight: endpoint, stable plateau, equilibrium line if present
speaker sentence: 这张图最重要的是说明催化剂不是短时高活性，而是在长时间运行中仍保持稳定。
```

### Spectroscopy figure

```text
key trend: peak appears, disappears, shifts, or changes intensity
key numbers: peak position, coordination number, energy shift
what to highlight: only the diagnostic peak or peak region
speaker sentence: 这里要看的不是整张谱图，而是该特征峰的变化是否支持位点环境变化。
```

### Microscopy figure

```text
key trend: isolated sites, clusters, morphology change, or size confinement
key numbers: scale bar, particle size, distance
what to highlight: visible clusters / absence of clusters / confined region
speaker sentence: 显微图提供的是直接观察证据，但仍要与光谱或计算证据共同理解。
```

### Table figure

```text
key trend: this work differs from representative literature in lifetime, selectivity, STY, or conditions
key numbers: only 2-4 most important values
what to highlight: this-work row and selected comparison rows
speaker sentence: 表格不用逐项读，重点是本文在相同或更苛刻条件下兼顾了哪些指标。
```

## Caution

If a trend is indirect, use cautious wording such as `支持`, `提示`, `与...一致`. Do not write `证明` unless the evidence is direct and strong.
