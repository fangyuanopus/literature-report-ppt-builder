# Final Scoring Rubric

Use this rubric after applying quality gates to produce a concise self-review of the final deck.

## Score Dimensions

Score each dimension from 0 to 10:

```text
paper logic clarity:
main-paper + SI integration:
figure source integrity:
figure-claim alignment:
slide readability:
sample-deck style fidelity:
terminology consistency:
claim calibration:
speaker readiness:
delivery completeness:
```

## Interpretation

```text
90-100: presentation-ready, only minor polishing needed
80-89: strong deck with several fixable issues
70-79: usable draft but needs revision before presentation
<70: do not present without significant revision
```

## Required Output Format

Include this in `quality_check_report.md`:

```text
final_quality_score: __/100
strongest aspects:
- ...
main issues:
- ...
recommended fixes:
- ...
unresolved limitations:
- ...
```

## Scoring Rules

- Never give a high score if scientific visuals are not traceable.
- Penalize overclaiming even if the slide looks good.
- Penalize slides that cannot be read in presentation mode.
- Penalize missing SI review when SI was provided.
- Reward clear page claims and good evidence sequencing.
